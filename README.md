# CLOX

This is my interpretation of the clox language defined by Robert Nystrom
in his book: [Crafting Interpreters](https://craftinginterpreters.com/).

clox is written in c and is interpreted by a virtual machine

Previously I wrote a version of his jlox language from the same book.
My version was written in TypeScript and hence is called
[tlox](https://github.com/keesschotanus/tslox).


## My version of clox

I will follow Robert's implementation as close as possible,
but I do want to replace the collection classes he uses
with my version of [collections](https://github.com/keesschotanus/collections).
